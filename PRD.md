# 产品需求文档 (PRD) - 朋友圈 MBTI 社交分析 H5

## 1. 项目概况
- **项目名称**：朋友圈 MBTI 社交探测器 (Friendship & Business MBTI Analyzer)
- **项目背景**：用户在社交和商务场景中，往往难以快速准确地了解对方的性格特点和潜在需求。通过分析对方的朋友圈截图，利用 AI 的多模态能力和心理学分析（MBTI），帮助用户快速建立认知。
- **核心目标**：通过分析 5 张以上的朋友圈截图，识别对方的 MBTI 性格，并提供针对性的社交/商务建议，帮助用户与目标人物建立更深层的联系。

## 2. 目标用户
- **社交需求者**：希望通过朋友圈了解潜在朋友、约会对象的兴趣爱好和性格，提高沟通效率。
- **商务/销售人员**：希望通过分析客户的朋友圈，精准把握客户偏好，将其发展为长期客户或合作伙伴。

## 3. 核心功能模块

### 3.1 截图上传模块
- **功能描述**：用户上传 5 张或以上（建议 5-10 张）的对方朋友圈截图。
- **要求**：
  - 支持多图上传（微信 H5 环境适配）。
  - 图片预览与删除。
  - 至少 5 张的硬性校验。

### 3.2 AI 深度分析模块 (核心)
- **使用的模型**：火山引擎豆包大模型 `doubao-seed-1.6-thinking`。
- **分析维度**：
  - **MBTI 识别**：根据朋友圈内容（文字、配图风格、发布频率、话题偏好）推测其 MBTI 类型。
  - **性格画像**：总结对方的性格关键词（如：热爱生活、事业心强、内敛温和等）。
  - **兴趣图谱**：提取对方关注的领域（如：运动、科技、美食、摄影等）。

### 3.3 社交/商务建议模块
- **功能描述**：基于分析结果生成实操建议。
- **内容包括**：
  - **破冰话术**：针对不同性格设计的开场白。
  - **雷区预警**：与其交往中应避免的话题或行为。
  - **转化策略**：如何将其引导至商务合作或深度友谊。

## 4. 用户路径 (User Flow)
1. **进入页面**：简约现代的 H5 首页，明确展示产品价值。
2. **上传截图**：点击上传按钮，选择 5 张以上截图。
3. **分析中状态**：展示加载动画，文案提示“AI 正在阅读朋友圈内容，深度解析中...”。
4. **查看报告**：展示精美的 MBTI 分析报告，包括雷达图、性格标签、实操建议。
5. **保存/分享**：支持生成长图保存到本地。

## 5. 技术架构
- **前端**：Vue 3 / React (H5 适配)，Tailwind CSS 或 Vant UI（移动端组件库）。
- **后端**：Node.js (Express/NestJS) 或 Python (FastAPI/Flask)。
- **AI 接口**：火山引擎方舟平台 API (Ark API)。
- **部署**：支持 HTTPS 访问的云服务器。

## 6. 风险与合规
- **隐私声明**：页面显著位置标注“仅用于分析，不存储用户上传的图片，保护隐私”。
- **免责声明**：AI 分析结果仅供参考，不代表绝对准确的心理学诊断。

## 7. 迭代计划
- **V1.0**：实现基础的上传、MBTI 分析、社交建议功能。
- **V1.1**：增加“情商话术库”，支持针对特定场景（如：求职、表白）生成回复建议。
- **V1.2**：增加性格匹配度测试（用户也上传自己的截图，看两人是否合拍）。
